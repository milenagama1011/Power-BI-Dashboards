# Dashboard 1 – SMEs Digital Marketing Performance KPIs

Live Dashboard:
https://app.powerbi.com/reportEmbed?reportId=d20f06b9-b494-4f64-87c8-9197600e39ec

## Overview
Interactive dashboard analyzing digital marketing performance for SMEs. Supports budget optimization and campaign effectiveness analysis.

## Data Sources
Excel / CSV datasets transformed using Power Query and Power BI data modeling.

## Key Metrics
• Total Engagement  
• Total Ad Spend  
• Audience Reach  
• Cost per Engagement (CPE)  
• Campaign Success Rate  

## Key Features
• Industry and country filters  
• Channel comparison  
• Drill-down by company size  
• Budget sliders  
• Time-series performance trends  

## Business Impact
Helps marketing teams identify the most cost-effective channels, optimize budget allocation, and improve ROI through data-driven decisions.

# Dashboard 2 – FIFA 2023 Scouting & Recruitment Insights

Live Dashboard:
https://app.powerbi.com/reportEmbed?reportId=b0e43b79-2567-478a-981f-e56a864d27d8

## Overview
Interactive scouting analytics dashboard for player recruitment decisions. Evaluates performance, potential, contracts, and salary trends.

## Data Sources
Structured player datasets modeled using Power Query and DAX.

## Key Metrics
• Player Overall Rating  
• Potential Development  
• Market Value  
• Weekly Wage  
• Contract Expiry Timeline  

## Key Features
• Position, age, and nationality filters  
• Player potential vs performance scatter analysis  
• Club comparison views  
• Market value vs wage relationships  
• Contract expiration forecasting  

## Business Impact
Supports data-driven recruitment and scouting decisions by identifying high-potential players and optimizing transfer investments.
