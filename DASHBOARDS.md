# Dashboard Catalog

## Dashboard 1: Academic Performance & Engagement Dashboard
### Overview
Interactive dashboard tracking programme performance and student engagement across 45 academic programmes to support institutional leadership decisions and resource planning.
### Access Dashboard
https://app.powerbi.com/reportEmbed?reportId=d20f06b9-b494-4f64-87c8-9197600e39ec&autoAuth=true&ctid=1eb34f73-38dd-42db-98db-53b54e749d52
### Key Metrics Tracked:
- **Participation Rates** - Overall student enrollment and attendance metrics
- **Completion Rates** - Programme completion and graduation statistics
- **Engagement Indicators** - Student interaction and involvement levels
- **Programme Performance Comparisons** - Relative performance across all 45 programmes
- **Trends Over Time** - Historical performance patterns and trajectory analysis
### Data Sources
- Excel spreadsheets
- Institutional datasets
- Power Query transformations for data cleansing and enrichment
### Business Impact
✅ Enabled leadership to quickly identify underperforming areas  
✅ Prioritized targeted improvements based on data insights  
✅ Supported data-driven policy decisions  
✅ Streamlined resource allocation across programmes

### Overview
Data storytelling dashboard analyzing team and player performance statistics to demonstrate advanced visualization, filtering, and interactive reporting techniques. This showcase highlights the ability to create engaging, user-friendly analytics for non-technical audiences.
### Access Dashboard
[**[Open Dashboard](https://app.powerbi.com/reportEmbed?reportId=b0e43b79-2567-478a-981f-e56a864d27d8)**](https://app.powerbi.com/reportEmbed?reportId=b0e43b79-2567-478a-981f-e56a864d27d8&autoAuth=true&ctid=1eb34f73-38dd-42db-98db-53b54e749d52&actionBarEnabled=true&reportCopilotInEmbed=true)
### Key Metrics Tracked
- **Goals & Assists** - Individual and team scoring performance
- **Team Rankings** - Competitive standings and performance hierarchy
- **Match Outcomes** - Win/loss records and performance patterns
- **Player Performance Stats** - Individual player statistics and contributions
### Data Sources
- Public FIFA tournament datasets
- Excel data files
- Power Query transformations and data cleaning
### Business Impact
✅ Demonstrates advanced Power BI storytelling capabilities  
✅ Showcases user-friendly dashboard design for non-technical audiences  
✅ Illustrates best practices in data visualization and UX  
✅ Proves ability to communicate complex data in an engaging manner
